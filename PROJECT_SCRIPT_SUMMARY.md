# Ringkasan Script Project TukangDekat

`Setup_tukangdekat_github.sh` adalah script project utama.

## Hasil Cek

- Jumlah issue di repo ini: **20**
- Jumlah item di project board: **30**
- Jumlah item project yang berasal dari repo ini: **2**

## Kesimpulan

Project board yang dipakai bersifat **multi-repo**. Jadi jumlah item di board tidak sama dengan jumlah issue di repo ini.

Dari hasil cek, hanya **2 item** yang benar-benar berasal dari repo `PM_UAS_rekayasa_Sistem_Informasi`, sedangkan sisanya berasal dari repo lain yang juga ikut masuk ke project board.

## Catatan

`Setup_tukangdekat_github_fajar1180.sh` hanya versi yang dipersempit ke scope `Fajar1180`.

Kalau perlu, file ini bisa dipakai sebagai catatan cepat untuk menjelaskan kenapa jumlah issue dan jumlah item project berbeda.
